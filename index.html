Twitter Analysis Dashboard

This is a dashboard application built using Java Enterprise Edition and designed using Google's material design. The application fetches tweets based on user query from various domains and does sentiment analysis by calling a python script underneath. Later it displays the results using Chart.js. It also supports stream analysis where Spark Streaming is used.

Technologies: JEE, Python, Apache Spark, AJAX, Apache Tomcat
Posted by Bijoyan Das on August 3, 2017
